Ten projekt ma wyłącznie charakter kolekcjonerski i rozrywkowy.
Nie jest w żaden sposób powiązany z żadną instytucją państwową.
Generowane materiały nie są dokumentami, nie posiadają żadnej mocy prawnej i nie mogą być używane do identyfikacji ani jakichkolwiek czynności urzędowych.
Wszelkie dane używane w generatorze mają charakter fikcyjny lub powinny być wprowadzane wyłącznie w celach testowych/kolekcjonerskich.
